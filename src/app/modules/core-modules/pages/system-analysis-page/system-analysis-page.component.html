System Analysis

SQL Mantra Tools System Analysis Module will analyse how the system is configured and performing and will provide best practice recommendation to remedy any defects. It will also analyse the performance KPI recorded by the Performance Logging module and will summarise the data in an easy to understand user friendly interfaces.

The following areas are analysed:
  • SQL Server Setup Analysis
  • Tempdb Setup Analysis
  • Server Memory Analysis
  • CPU Utilisation Analysis
  • SQL Server Wait Analysis
  • Maintenance Job Analysis
  • Database Setup Analysis
  • Scheduler Job Analysis
  • Deadlock Analysis
  • Database File Layout Analysis
  • Disk Latency Analysis
  • Database Size History Analysis
  • Table Size Analysis
  • Index Usage Analysis
  • Slow Query Analysis
  • Blocking Analysis
  • Dynamics NAV Setup Analysis
